Performance and Dependability

JTL has been used for supporting round-trip engineering processes involving performance and dependabily analysis (e.g, Petri Nets, Queueing Networks, ...) of software system models.

  • UML to Queuing Networks: the case study refers to the UML to QN transformation, which relates software architectural models (UML-MARTE models) to performance models (Queuing Networks).
